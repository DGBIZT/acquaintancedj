# Знакомство с Django
## Создание проекта интернет магазина

1. Описание:
```
 Cоздание проекта интернет-магазина, который будете дорабатывать на каждом уроке в течение всего курса.
```
2. Клонируйте репозиторий:
```
https://github.com/DGBIZT/acquaintancedj.git
```
3.
###### №1
```
1. Настройка проекта.
2. Создание и настройка приложения
3. Создание шаблонов
    * home.html
    * contacts.html
4. Реализация контроллеров
```
4.
###### №2
```
1. Подключение СУБД PostgreSQL
    * Создание БД
    * Внесение изменений в настройки подключения в файле setting.py
2. В приложении каталога созданы модели Product и Category и описаны для них базовые настройки.
      Product:
        * наименование,
        * описание,
        * изображение,
        * категория,
        * цена за покупку,
        * дата создания,
        * дата последнего изменения.
      Category:
        * наименование,
        * описание.
3. Перенесено отображение моделей в базу данных с помощью инструмента миграций. 
      Для этого:
        * созданы миграции для новых моделей;
        * применены миграции.
4. - Создание суперпользователя.
   - Зарегистрированы модели Product и Category в админке.
   - Настроены отображение для моделей:
        * Для Category выведены id и name в списке.
        * Для Product выведены id, name, price и category в списке.
        * Настроены фильтрации продуктов по категории.
        * Настроены поиск по полям name и description
5. Через инструмент shell заполнен список категорий, а также выберите список категорий, применив произвольные рассмотренные фильтры.
    1) Вход в Django shell
    2) Создано несколько категорий и продуктов.
    3) Выполнены следующие запросы:
        * Получите все категории.
        * Получите все продукты.
        * Найдены все продукты в определенной категории.
        * Обновление цены для определенного продукта.
        * Удалите продукта.
    4) Все решения сохранены в папке media/screenshots в виде скриншотов.
```
